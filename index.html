Introduction

Welcome to K2S API Guide.

This Guide is supposed to clarify the way how you can work with K2S Cloud Storage.

Using the following documentation you can automate the most often actions with K2S Cloud Storage such as

  • Download files

  • Upload files

  • Authorize on the K2S Service

  • integrate a third party applications

  • create an automation scripts

and et cetera

Authorization

Authorization without Captcha

CURL:

Request:
$ curl -X POST\
-H "Content-type: application/json"\
'{"username":"yourbox@gmail.com","password":"yourpassword"}'\ 
https://keep2share.cc/api/v2/login
Response:
{
"status":"success",
"code":200,
"auth_token":"flmuf4o3uo59s105546dmvmq60"
}

PHP:

<?php
include "Keep2ShareAPI.php";
$api = new Keep2ShareAPI("yourbox@gmail.com", "yourpassword");
$result = $api->login();
var_dump($result);
/*
bool(true)
*/
var_dump($api->getAuthToken());
/*
string(flmuf4o3uo59s105546dmvmq60)
*/

Console Uploader

The application allows automated and quickly upload files to your account and receive links.

There are things it can do for you:

  • Upload a specific file from your computer
  • Upload files from specific folder from your computer
  • Create the CSV file with operetion's result.

Quick start

  • Download the application for your operating system: Windows MacOs Linux
  • Extract the archive. Go to the folder with the application.
  • Get access_token
  • Usage

Usage

$ partnercli upload

Options

-a, --access-token=access-token              (required) access token from your profile
-c, --path-to-csv=path-to-csv                [default: upload_log.csv] local path to generated result file
-d, --destination-folder=destination-folder  folder Id on server
-j, --stdout-json=stdout-json                stdout in json format
-n, --domain-name=k2s|fb|p2m|tz              [default: k2s] domain name for generated URLs
-s, --source-folder=source-folder            (required) path to local folder

Aliases

$ partnercli u
$ partnercli upl

Examples

\partnercli-win.exe upload 
-a 23ffbcbacb549ae531a45d04b32880bc8560a34a -n k2s -s .\
